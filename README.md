# Homelab Project
Documentation overview of a home server building project and skills learned through tinkering and application.

## Hardware Specs
* **System:** Dell Optiplex 3010
* **OS:** Ubuntu Server
**Processor:** Intel(R) Core(TM) i3-3220 CPU @ 3.30GHz
**Memory:** 16GB DDR3 1600MHz
**Storage:** 120GB Kingston SSD

## Goals
- Learn Linux environment and administration
- Familiarize with deploying with Docker
- Learn networking
- Build a cybersecurity lab to develop skills
- Tinker with automation and convenient user services
